## 화면 종류

각 화면은 화면 구성을 엄격히 따른다.

| ID | 화면 | 진입 경로 | 화면구성 |
|---|------|------|------|
| s1 | 카메라 화면 | 앱 진입 시 기본 화면 | 화면구성1 |
| s2 | 가이드라인 추출 화면 | s3 촬영 완료 후 '완료' 탭 / s4 사진 선택 후 '완료' 탭 | 화면구성2 |
| s3 | 가이드라인용 사진 촬영 화면 | s1 '+' 버튼 모달에서 '가이드 사진 찍기' 탭 | 화면구성1 |
| s4 | 갤러리 화면 | s1 '+' 버튼 모달에서 '가이드 사진 등록하기' 탭 | 화면구성3 |

## 화면 전환 요약

| 출발 | 조건 | 도착 |
|------|------|------|
| s1 | '+' → '가이드 사진 찍기' | s3 |
| s1 | '+' → '가이드 사진 등록하기' | s4 |
| s2 | 완료 버튼 탭 | s1 (새 가이드라인 적용) |
| s2 | 취소 버튼 탭 | s1 (가이드라인 미적용) |
| s3 | 취소 버튼 탭 | s1 |
| s3 | 재촬영 카드 → 재촬영 버튼 탭 | s3 |
| s3 | 재촬영 카드 → 완료 버튼 탭 | s2 |
| s4 | 취소 버튼 탭 | s1 |
| s4 | 완료 버튼 탭 | s2 |
