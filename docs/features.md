타겟: 여자친구의 사진을 찍어주는 데 어려움을 겪는 20~30대 남성과 그들의 여자친구

가설: 사진을 찍어줄 때 가이드라인이 있으면 사진을 더 잘 찍어줄 것이다.
솔루션: 카메라 위 가이드라인 제공.


사용 흐름

1. 기존에 잘 찍었던, 혹은 구도가 마음에 들었던 사진을 등록하면 인물을 감지해 가이드라인이 따지고, 카메라 위에 그 가이드라인이 띄워지며 그에 맞게 인물이 들어왔을 때 가이드라인 색이 변한다.
2. 등록돼있는 가이드라인을 누르면, 해당 가이드라인이 카메라 화면 위에 띄워진다. 그에 맞게 인물이 들어왔을 때 가이드라인 색이 변한다.


MVP 기능

1. 기본 사진 찍기 기능
    - 저장 위치: 사용자 디바이스
2. 가이드라인을 위한 사진 등록
    - 디바이스에 있는 사진 등록, 혹은 앱 내에서 사진 찍기
3. 인물 감지
    - 감지 실패 시: 에러 메시지
4. 라인 따기
    - 한 번 등록된 가이드 라인은 사용자가 삭제하기 전까지 유지됨
5. 카메라 화면에 가이드 오버레이 띄우기
6. 가이드에 맞는지 확인
