## 컴포넌트

각 컴포넌트는 모든 화면에서 같은 스타일을 유지함.

| 컴포넌트 | 내용 | 화면 | 위치 | 활성화 조건 |
|------|-----------|------|------|------|
| 촬영 버튼 | 사진 촬영 | s1, s3 | 카메라 영역 하단 중앙 | 항상 |
| 갤러리 버튼 | 디바이스 갤러리 열기 | s1 | 조작 영역 좌측 하단 | 항상 |
| 전면/후면 전환 버튼 | 전면/후면 카메라 전환 | s1, s3 | 조작 영역 우측 하단 | 항상 |
| 화면 비율 조작 버튼 | 카메라 화면 비율 변경 | s1, s3 | 조작 영역 중앙 | 항상 |
| 가이드라인 오버레이 | 선택한 가이드라인을 카메라 영역 위에 오버레이로 표시 | s1 | 카메라 영역 전체 | 가이드라인 목록에서 요소 선택 시 |
| 가이드라인 목록 | 가로 스크롤 리스트. 목록 배경 화면 X(요소만 표시). 초기 상태(가이드라인 없음)엔 '+' 버튼만 표시. 가이드라인 있음: 썸네일 목록 + 오른쪽 끝에 '+' 버튼 | s1 | 촬영 버튼 상단 | 항상 |
| 가이드라인 요소 | 탭: 카메라 영역에 오버레이 적용. 길게 누르기: '삭제하시겠습니까?' 확인 모달 표시 | s1 | 가이드라인 목록 내 | 가이드라인 1개 이상 등록 시 |
| '+' 버튼 | 탭 시 모달 팝업 표시. 모달 내 '가이드 사진 찍기' 탭 → s3 이동. '가이드 사진 등록하기' 탭 → s4 이동 | s1 | 가이드라인 목록 오른쪽 끝 | 항상 |
| 프로세스 바 + 로딩 애니메이션 | 가이드라인 추출 진행 중 표시 | s2 | 화면 중단 | 추출 진행 중 |
| 가이드라인 미리보기 | 사진 위에 흰색 선으로 가이드라인 표시. MVP: 수정 불가 | s2 | 사진 영역 위 오버레이 | 추출 성공 후 |
| 에러 메시지 | 가이드라인 추출 실패 시 표시 | s2 | 화면 중단 | 추출 실패 시 |
| 완료 버튼 | 추출된 가이드라인 저장 후 s1으로 이동. s1 복귀 상태: 새 가이드라인이 목록에 추가되고 적용된 상태 | s2 | 상단바 우측 | 추출 성공 후 |
| 취소 버튼 | 추출 데이터 삭제 후 s1으로 이동. s1 복귀 상태: 가이드라인 미적용 | s2 | 상단바 좌측 | 항상 |
| 뒤로가기 버튼 | s1으로 이동 | s3 | 조작 영역 좌측 하단 | 항상 |
| 재촬영 카드 | 촬영 버튼 탭 후 표시되는 모달. 재촬영 버튼: s3 초기 상태로 복귀. 완료 버튼: 촬영한 사진을 가지고 s2로 이동 | s3 | 모달 팝업 | 촬영 버튼 탭 후 |
| 사진 그리드 | 그리드 레이아웃. 단일 선택 | s4 | 갤러리 영역 | 항상 |
| 취소 버튼 | s1으로 이동 | s4 | 상단바 좌측 | 항상 |
| 완료 버튼 | 선택한 사진을 가지고 s2로 이동 | s4 | 상단바 우측 | 사진 1장 선택 시 (미선택 시 비활성화) |
